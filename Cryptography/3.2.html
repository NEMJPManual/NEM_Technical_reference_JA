署名と検証

メッセージMMM、秘密鍵kkk、対応する公開鍵AAAを所与のものとし、署名の作成のために以下の手順が取られます。

H(k)=(h0,h1,...,h511) H(k) = (h_0, h_1, ..., h_{511}) H(k)=(h​0​​,h​1​​,...,h​511​​)

r=H(h256,...,h511,M) r = H(h_{256}, ..., h_{511}, M) r=H(h​256​​,...,h​511​​,M) ただしカンマは結合を表す。 R=rB R = rB R=rB S=(r+H(R,A,M)a)modq S = (r+H(R,A,M)a) \: mod \: q S=(r+H(R,A,M)a)modq

この時、(R,S)(R,S)(R,S)を秘密鍵kkkによるメッセージMMMの署名と呼びます。 S<qS < qS<qかつS>0S > 0S>0を満たす署名のみが有効であることに注意してください。 これにより署名の展性(malliability)の問題を防ぐことが可能になります。

特定のメッセージMMMの署名(R,S)(R,S)(R,S)の、公開鍵AAAに対する有効性をチェックする場合、 まずS<qS < qS<qかつS>0S > 0S>0をチェックしたのち以下を計算し、

R~=SB−H(R,A,M)A \tilde{R} = SB - H(R,A,M)A ​R​~​​=SB−H(R,A,M)A

そして以下の等式が成り立つことを確認します。

R~=R \tilde{R} = R ​R​~​​=R

Sが上記の方法で計算されていれば

SB=rB+(H(R,A,M)a)B=R+H(R,A,M)A SB = rB + (H(R,A,M)a)B = R + H(R,A,M)A SB=rB+(H(R,A,M)a)B=R+H(R,A,M)A

が成り立つのでR~=R\tilde{R} = R​R​~​​=Rの確認が有効なことがわかります。
